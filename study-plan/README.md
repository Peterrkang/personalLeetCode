# 3-Month Interview Prep Plan — Algorithms + System Design

**Target:** Mid / Senior SWE roles
**Language:** JavaScript / TypeScript
**Pace:** Full-time (~6+ hrs/day, 6 days/week)
**Start:** 2026-06-23 → **Target finish:** 2026-09-22 (~13 weeks)

> One rest/buffer day per week (suggest Sunday). Use it lightly — review notes, no new material. Burnout kills 3-month plans faster than any hard problem.

---

## The daily rhythm (~6.5 hrs)

| Block | Time | What |
|-------|------|------|
| **1. Algorithms — learn** | 90 min | New pattern/topic of the day. Read, watch, take notes, do 1 worked example by hand. |
| **2. Algorithms — practice** | 120 min | 3–4 problems on that pattern. Time-box each (25–35 min) then read the solution. |
| *Break* | 30 min | Walk. Don't skip. |
| **3. System design** | 90 min | One concept or one case study. Draw the diagram, write down tradeoffs. |
| **4. Review / mock / behavioral** | 60 min | Re-solve 1 problem from memory, or spaced-repetition flashcards, or behavioral stories. |

**Weekly extras (rotate into Block 4):**
- 1 timed mock per week from week 3 onward (45 min, no IDE help). 2/week from week 9.
- Maintain a **mistakes log** — every bug/wrong approach goes in it. Re-read it weekly.

---

## The three phases

### Phase 1 — Foundations (Weeks 1–4)
Rebuild core data structures & algorithm patterns. System design vocabulary & building blocks. Goal: stop getting *stuck*, start recognizing patterns.

### Phase 2 — Depth (Weeks 5–8)
Harder patterns (graphs, DP, intervals, tries). Real system design case studies. Goal: solve mediums comfortably, design a system end-to-end.

### Phase 3 — Polish & simulate (Weeks 9–13)
Mixed/random problem sets, hards, 2 mocks/week, behavioral, deep system design for senior signal. Goal: perform under interview pressure.

## How to use it
1. Each morning, check the day's plan and topics.
2. Pull the day's topics from the two curriculum files.
3. Log every mistake. Re-solve flagged problems on your buffer day.
4. Friday: short self-review — what clicked, what didn't, adjust next week.
