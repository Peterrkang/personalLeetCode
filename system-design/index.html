System Design — The Course

A full system-design interview course modeled on Design Gurus' Grokking the System Design Interview: building blocks first, then the classic case studies, then senior-level distributed-systems depth. One page per week; read it, then rewrite the key ideas in your own words — teaching it back is how it sticks.

Pairs with the week-by-week SD Curriculum (the schedule) and the Flashcards (decks SD1–SD12, spaced-repetition review of every topic here). Read a page → drill its deck until mastered.

How to study each week

  1. Read the page (~30 min), drawing every architecture yourself as you go.
  2. Do the exercise at the bottom on paper, timed, using the 6-step framework.
  3. Run the self-check — anything you can't answer out loud, reread.
  4. Drill the matching SD deck in Flashcards until every card reaches box 5.
